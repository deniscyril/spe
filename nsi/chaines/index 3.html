    Variables E03: Chaînes de caractères

    A retenir

    • But: Python permet de manipuler les lettres, mots, phrases, texte ... Ces données sont du type chaine de caractère notée String.
    • Comment: Une chaîne de caractère est un ensemble de caractères placés entre guillemets " ou '.
    • A savoir: La position de chaque lettre d'une chaîne est repérée par un indice. La première lettre a pour indice 0, la suivante 1, etc.
    • Exemples d'affectation: 
    1
2
3
    prenom = "Cyril"
citation = "L ordinateur a de la mémoire mais aucun souvenir"
lettre = 'a'

    Objectifs

    type chaine, concaténation, conversion de type

    Manipulation de chaînes

    1. L'exemple ci-dessous utilise l'instruction type. Recopie le programme et vérifie que le type de la variable ma_chaine est ce qui signifie du type chaine de caractère (str pour sting).

      ma_chaine = "NSI"
type(ma_chaine)
    2. Dans la console écrit:
      En mode console
      a = "Je"

      Ajoute les chaînes a et b. Pour cela écrit dans la console:
      En mode console

    3. Quel résultat obtiens-tu ? On dit que les chaînes a et b ont été concaténées.
    4. Stocke le résultat de a+b dans une variable chaine_1.
    5. Essaye d'ajouter chaine_1 à nb. Relève le message d'erreur qui apparaît. Explique cette erreur en utilisant la notion de type de variable.

    6. Saisie le code suivant:

type(chaine_nb)
      Explique pourquoi dit-on que l'instruction str() permet de réaliser une conversion de type.

    Activité 2: Affichage de chaines

    Découverte

    Jusqu'ici pour connaître le contenu d'une variable, on a saisi son nom dans la console puis appuyer sur entrée. On peut aussi utiliser une instruction pour afficher dans la console le contenu d'une variable ou une chaine de caractère.

    print et f-string

    1. Saisie dans la console python le code suivant: 
      c1 = "bonjour"
c2 = ' le monde !'
print('hello')
print(c1 + c2)
      Expliquer ce que fait l'instruction print.
    2. Modifie la dernière ligne du script afin d'affiche: bonjour bonjour le monde !

    fstrings sont des chaînes de caractère plus complexes mais très utiles pour afficher des messages contenant des informations variables.

    1. Saisie dans la console python le code suivant:
      En mode console
      prof = "M.DENIS"
matiere = "NSI"
nb_eleve = 19
msg = f"{prof} enseigne la {matiere} a {nb_eleve} élèves de première."
print(msg)
    2. Rédige l'exécution ligne à ligne de ce script.
    3. Modifie ce code pour afficher M.MARTIN enseigne la Mathématique a 35 élèves de première.
    4. Ajoute une nouvelle variable niveau qui contient la chaine "terminale" afin d'afficher: M.MARTIN enseigne la Mathématique a 35 élèves de terminale.
    5. Selon toi, qu'est-ce qui rend les f-strings plus pratiques que les chaînes de caractères classiques ?
