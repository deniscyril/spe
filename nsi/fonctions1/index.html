    Fonctions E01

    Présentation

    A retenir

    Reconnaître une fonction:

    • Une fonction se reconnaît par le mot clé def suivi du nom de la fonction.
    • Entre les parenthèses on peut trouver aucun, un ou plusieurs arguments.
    • Les instructions exécutées par la fonction sont indentées après le mot clé def.
    • Lorsque la fonction renvoie une valeur, elle le fait en utilisant l'instruction return

    Pourquoi utiliser une fonction ?

    1. Lorsqu'un bloc d'instructions est utilisé plusieurs fois dans le programme. On écrit une fonction qui contient ces instructions et on appelle la fonction plutôt que de copier/coller du code identiques. Cela s'appelle Factoriser le code.
    2. Python ne contient qu'une trentaine d'instructions. En écrivant des fonctions on définit de nouvelles instructions adaptées au problème à résoudre.
    3. Certaines fonctions ont déjà été écrites par d'autres programmeurs et sont mises à votre disposition, vous pouvez les importer afin de les utiliser directement. C'est le cas de randint qui permet de tirer un nombre aléatoirement.

    # Déclaration de la fonction bonjour_v2
def bonjour_v2(qui):
    """
    In: reçoit la chaine de caractères qui
    Travail: Affiche Bonjour suivi de qui
    Out: Rien
    """
    print('Bonjour ', end="") #end="" permet de ne pas revenir a la ligne
    print(qui)
# Appels de la fonction bonjour_v2
bonjour_v2("Toto !")
nom = "Monsieur DENIS"
