    Structures Conditionnelles E03

    A retenir

    On étudie ici une structure très courante qui se décompose en 2 tests conditionnels et 3 blocs d'instructions dont 1 seul sera exécuté.

    La structure générale est:

    if condition_1 == True: 
    bloc 1
elif condition_2 == True:
   bloc 2
else: 
   bloc 3

    Attenion Syntaxe: Ne pas oublier le : à la fin des lignes contenant if, elif et else. Remarquez aussi l'indentation des 3 blocs.

    Activité 3: Ecriture de code

    Situation 1

    Un nombre pair est divisible par 2. Une autre manière de le dire est que le reste de la division par 2 vaut 0.

    • Exemple 14 est pair car lorsqu'on le divise par 2 il reste 0 (et le quotient vaut 7). En Python, pour obtenir le reste de la division de 14 par 2, on écrit: 14 % 2

    En mode console
        14 % 2
    5 % 2
    4 % 2
    3 % 2
    13 % 2
    1. Tester ce code dans une console python en observant les valeurs renvoyées. 2. Expliquer comment avec l'opérateur % on peut en python tester si un nombre est pair ou impair. 3. Ecrire un programme qui utilise une variable n contenant un entier et affiche pair si celui-ci est pair et impair dans le cas où il est impair.

    Situation 2

    Une année est bissextile si elle est divisible par 4. Cependant les siècles ne sont pas bissextiles, sauf les multiples de 400. La valeur de l'année est stockée dans une variable notée: an. Ecrire un programme qui affiche si une année est bissextile. Vérifiez que 2100 n'est pas bissextile contrairement à 2012.

    Aide

    Pour savoir si l'année est divisible par 4 on utilisera an % 4 et on regarde si cela vaut 0
