    Premières animations

    Comment Animer ?

    Comme pour un dessin animé, on affiche une succession d'images fixes. La stratégie est:

    1. Dans update(), on modifie les valeurs des variables dont dépend l'affichage.
    2. Dans affichage() on place toutes les instructions qui réalisent les dessins.

    Exemple

    import pyxel
pyxel.init(128,128,title="Rectangle")

def update():
    global position_x
    if position_x >128:
        position_x=0
    position_x = position_x + 1

def draw():
    pyxel.cls(0)
    pyxel.rect(position_x,20,10,5,2)

position_x = 1
pyxel.run(update, draw)

    1. Que représente la variable: `position_x` ?
2. Que permet la ligne `position_x = position_x + 1` ?
3. Pourquoi faire un test conditionnel dans la fonction `update()` ?
4. Que permet l'instruction `global` ? Aurait-on pu faire autrement ?
5. Modifie le programme afin que le rectangle rebondisse sur le bord droit et gauche de l'écran.

    import pyxel
pyxel.init(128,128,title="Rectangle")

def update():
global position, vitesse
if position > 128 or position <0:
vitesse = - vitesse
position += vitesse


def draw():
pyxel.cls(7)
for i in range(0,129,8):
pyxel.line(0,position,128,i,1) 

#Programme Principal
# Initialisation des variables 
position = 0
vitesse = 1
pyxel.run(update, draw) #Appel en boucle les fonctions update et draw
